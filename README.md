# PDF Merger & Splitter Tool

A comprehensive desktop application for merging and splitting PDF files with an intuitive GUI.

## Features

### PDF Merger
- Merge multiple PDF files into a single PDF
- Drag and drop support for easy file addition
- Reorder files using Move Up/Down buttons
- View file information (page count, size)
- Progress indication during merge process

### PDF Splitter
- Split PDF into individual pages
- Extract specific page ranges (e.g., 1,3-5,7)
- Split by custom page ranges (e.g., 1-3,4-6,7-10)
- View split progress and status logs
